Закинь в этот проект все файлы с которыми работает программа

перед работой надо ввести в консоли проекта
git pull
когда закидываешь изменения в консоль вводишь
git add . (смотрит все изменения)
git commit -m "какой угодно текст" (создает коммит, в ковычках твой комментарий)
git push    (закидывает изменения)
